## 3. Authentication & Authorization

### 3.1 Login/Register

* Email + password
* Google OAuth2 login

---

### 3.2 JWT + Refresh Token

* Access Token:

  * TTL: 15–30 phút

* Refresh Token:

  * TTL: 7–30 ngày

* Lưu:

  * Access Token → Redux (memory)
  * Refresh Token → HTTP Only Cookie

---

### 3.3 Flow

* Access token hết hạn:

  * FE gọi /auth/refresh
  * lấy token mới
  * retry request

---

### 3.4 OAuth2

* Login Google → lấy user info
* Nếu chưa tồn tại → tạo user
* Trả JWT hệ thống (không dùng Google token)
